quando fazemos o layout de um site, o navegador representa cada parte dele como o "box model". 

margin - espaçamento entre elementos

border - circulam o padding e o content
	border: 3px solid blue:
	border-top: 2px dotted green;
	border-right: 4px dashed pink
	(borda: tamanho estilo cor)
	   solid = sólida
	   dotted = pontilhada
	   dashed =  tracejada

	border-radius = permite arredondar os cantos de um elemento.
	border-radius:50% = transforma o elemento em circulo.
	
	border-radius: 30% 20% 30% 20%
	(border-radius: topo direita inferior esquerda)   
	   

padding = espaçamento entre o bloco e seu conteúdo
      .post {
	padding: 15px 10px 5px 0;
       }
	    15px - para o topo
	    10px = para a direita
	     5px = para a parte inferior
	       0 = para o lado esquerdo

   tbm pode ser:
      .post{
	padding-top: 15px;
	padding-right: 10px;
	padding- bottom: 5px;
	padding-left: 0;
       }
	
content - é o que o bloco representa


background - com essa propriedade podemos mudar a cor do fundo, mudar imagem e a posição dela.
